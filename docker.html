DOCKER

I. Các câu lệnh trong Docker

Lệnh Ý nghĩa Ví dụ
docker login  Đăng nhập với docker hub docker login
docker logout Đăng xuất tài khoản hiện tại docker logout

 

II.Các câu lệnh hay dùng  Docker image

STT Lệnh Ý nghĩa Ví dụ
1 docker pull ten_image Tải image có tên là http từ docker hub docker pull php:7.4-alpine
2 docker search -d ten_image Tìm kiếm các images docker search php:7.4-alpine
3 docker images Liệt kê các images đã tải về, lệnh này không có đối số truyền vào docker images
4 docker rmi -f ten_image Xóa image đã tải với tùy chọn -f docker rmi -f php:7.4-alpine
5 docker images --help Trợ giúp các câu lệnh liên quan đến docker images docker images --help

 

III. Các câu lệnh hay dùng Docker container

STT Lệnh Ý nghĩa   Ví dụ
1 docker run -d ten_image Tạo container chạy ngầm   docker run -d php:7.4-alpine
2 docker run -d -p 81:80 ten_image Tạo container chạy ngầm, ánh xạ port 81 của host với port 80 của container,   docker run -d -p 81:80 php:7.4-alpine
3 docker ps hoặc docker container ls Liệt kê ra các container đã tạo, lệnh này không cần đối số truyền vào   docker ps hoặc docker container ls
4 docker exec --help Trợ giúp liên quan đến docker exec     
5 docker exec ID_cua_contaier comand Chạy comand trong vùng làm việc của container     
6 docker exec -it ID_cua_contaier bash Thực hiện tương tác với container sau khi được tạo, lưu ý, cần sử dụng đụng ID của container mong muốn. Để thoát chế độ tương tác hãy ấn tổ hợp phím CTRL + p và CTRL + q   docker exec -it 3e6196a97a3f bash
7 docker stop ID_cua_contaier Tạm dừng hoạt động của container   docker stop 3e6196a97a3f
8 docker rm ID_cua_contaier Xóa container   docker rm 3e6196a97a3f
8 docker stop $(docker ps -a -q) && docker rm $(docker ps -a -q) Xóa tất cả các container, lệnh này không có đối số truyền vào. Lưu ý cần gõ 2 lệnh   docker stop $(docker ps -a -q) && docker rm $(docker ps -a -q)

 

IV. Các câu lệnh thao tác Docker compose

STT Lệnh Ý nghĩa Ví dụ
1 docker-compose up  Chạy các service docker-compose up 
2 docker-compose down Down các service docker-compose down
3 docker-compose build Build các service docker-compose build
4 docker-compose restart Restart lại các service docker-compose restart
5 docker-compose exec service_name bash Truy cập vùng làm việc docker-compose exec php bash
